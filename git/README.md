# GIT

If starting a project, create a `.gitignore` file and include the next configuration.
```yaml
################################################
# Logs
################################################
logs
*.log
npm-debug.log*
yarn-debug.log*
yarn-error.log*
lerna-debug.log*
# Diagnostic reports (https://nodejs.org/api/report.html)
report.[0-9]*.[0-9]*.[0-9]*.[0-9]*.json

################################################
# Cache
################################################
tmp
temp
.env
# Optional npm cache directory
.npm
# node-waf configuration
.lock-wscript
# parcel-bundler cache (https://parceljs.org/)
.cache
.parcel-cache
# Grunt intermediate storage (https://gruntjs.com/creating-plugins#storing-task-files)
.grunt
# Optional eslint cache
.eslintcache
# TypeScript cache
*.tsbuildinfo
# Microbundle cache
.rpt2_cache
.rts2_cache_cjs
.rts2_cache_es
.rts2_cache_umd
# FuseBox cache
.fusebox
# SASS/SCSS cache
.sass-cache

################################################
# Dependencies
################################################
# Dependency directories
node_modules
jspm_packages
# Bower dependency
# directory (https://bower.io/)
bower_components
# Snowpack dependency
# directory (https://snowpack.dev/)
web_modules
# Yarn 2
.yarn/cache
.yarn/unplugged
.yarn/build-state.yml
.yarn/install-state.gz
.pnp.*

################################################
# Output
################################################
# Common output folders
out
lib
dist
build
target
# Compiled binary addons
# (https://nodejs.org/api/addons.html)
build/Release
# next.js build output
.next
# nuxt.js build output
.nuxt
# vuepress build output
.vuepress/dist

################################################
# Test
################################################
# Directory for instrumented libs generated by
#jscoverage/JSCover
lib-cov
# Coverage directory used by tools like istanbul
coverage
# nyc test coverage
.nyc_output

################################################
# Serverless
################################################
.serverless/

################################################
# Runtime Data
################################################
pids
*.pid
*.seed
*.pid.lock

################################################
# Temp files
################################################
# MAC specific
.DS_Store
# Files are rejected files (usually hunks patches
# but not only).
*.rej
# Compressed files are needed sometimes to upload
# content, and accidentally added to the repo
*.tgz
*.zip
*.rar
*.7z
# These are backup files generated by rustfmt
**/*.rs.bk

################################################
# IDE/Text Editor config files
################################################
.vscode
.idea

################################################
# Include your custom configuration
################################################

################################################
# Include your custom exclusions configuration
################################################
```
